This site provides insights into oil and gas production across the United States.

The data allows users to:

  • View total productioni and average production per year.
  • Compare production by state, county, commodity, and disposition.
  • Explore offshore vs onshore production trends.
  • Access data directly via API endpoints for integration into other projects.

This website was built as a data visualization and API tool and is built using Django, Tailwind CSS, and Django Rest Framework.
